# Insta-Shop Screening Test -- GIN & Postgres

## CLONE

```shell
git clone git@github.com:ayoadeoye1/insta-shop-screening.git
```

## INSTALL PACKAGES

```shell
go get .
```

## START APP (with Make)

- hot reload
```shell
make dev
```

- start server
```shell
make start
```

## Access Documetation

- Open API Swagger
```
http://localhost:8080/swagger/index.html#/
```
